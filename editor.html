Basic Code editor

Available keyboard shortcuts:

  • Tab key to indent two spaces (with or w/o selection)
  • Ctrl-A to select all
  • Ctrl-C to copy
  • Ctrl-X to cut
  • Ctrl-V to paste
  • Ctrl-Z to undo
  • Ctrl-Shift-Z to redo

Copy the finished text and paste in the text input box. Check the "Raw HTML" checkbox if you want to paste in the raw html mode for better appearance of code.
