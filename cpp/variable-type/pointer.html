Pointer TypeÂ¶

  • A derived type to store the memory address of an lvalue

  • * symbol after type to make a derived type BaseType *ptr;

  • * operator to dereference a pointer *ptr

  • & operator to reference an lvalue &lvalue

  • syntax

1int value = 10;
2int *ptr1;
3ptr1 = &value;  // reference, now you can use *ptr1 as an alias of value
4(*ptr1)++;  // value increased by 1 and become 11
5cout << "The value is " << *ptr1 << endl;  // dereference
6// type cast to void * type to be displayed correctly
7cout << "The memory address is " << static_cast<void *>(ptr1) << endl;

  • Pitfalls

    • confusing pointer syntaxes with reference syntaxes
