********
List ADT
********
An ordered (not sorted) list like structure.

Data
====
An ordered list of data of the same type.

Behaviors
=========
+ Add/append/insert at any location
+ Remove at any location
+ Access at any location
+ Modify at any location
+ Check length

Implementation
==============
+ C array

  * Fast access/modify anywhere
  * Fast indexing
  * Slow add/insert/remove in the middle
  * Fixed maximum length

+ Linked list

  * Fast add/insert/remove anywhere
  * Slow indexing
  * Flexible length
