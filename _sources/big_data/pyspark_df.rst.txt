DataFrame
=========
+ External readings

+ The high-level abstraction in PySpark
+ Efficiently handle structured data
+ Based on RDDs but provides more structure and functionality
+ Characteristics

  * Distributed (data is distributed across multiple machines)
  * Immutable (once created, it cannot be changed)
  * Lazy evaluation (operations are not executed until a operation that
    requires shuffling is called)
